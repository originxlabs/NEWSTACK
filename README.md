# NEWSTACK — Global News Intelligence Platform

A product of **CROPXON INNOVATIONS PVT LTD**

## About NEWSTACK

NEWSTACK is a free AI-powered news aggregation platform that delivers real-time global and local news with:

- **AI-Powered Summaries** - Get concise, accurate summaries of news articles
- **Audio News** - Listen to news with text-to-speech (50 free plays/day)
- **Multi-Source Verification** - See coverage from multiple trusted sources
- **Place Intelligence** - Local news and insights based on your location
- **Trending Topics** - Stay updated with what's trending globally and locally

## Technology Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- PWA with offline support

## Development

```bash
npm install
npm run dev
```
